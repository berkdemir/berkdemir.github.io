❓Question
While defining the point connections, why do we have directions that we click? What should we do here?
Also, these lines are act as normal lines, so, when selecting the tunnel, by mistake, user assign springs to these also.

To assign springs, we use line support group. Change the local system as shown in the figure.

With this change, x and z direction becomes "tangential" springs, and y direction is normal spring. Tension for the Ky should be 0 to make it compression only spring.

However, the directions of the springs are important!

❓Spring Directions
The springs should be aligned same way. How?
There is ground outside, so it's compression only. We need to also make sure that we have compression only defined correctly.
